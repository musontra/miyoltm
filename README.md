


## 🚀 Yerelde Nasıl Çalıştırılır?

### 🔹 1. Gereksinimler

* Node.js (v18+ önerilir)
* npm veya yarn
* Android Studio (Android SDK kurulu olmalı)
* Java JDK 17+
* Emülatör veya fiziksel Android cihaz

---


---

### 🔹 3. Bağımlılıkları Kurma

```bash
npm install
```

---

### 🔹 4. Android İçin Build Alma

```bash
npm run build
npx cap sync android
npx cap open android
```

Android Studio açıldıktan sonra:

* **Build > Generate Signed Bundle / APK**
* Release APK oluştur
